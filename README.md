## ğŸ§© How It Works

### 1. Add a Task

Type your task into the input field and click **Add**.

### 2. Edit a Task

- Select a task in the Listbox
- Click **Load to Edit**
- Modify the text
- Click **Edit** to update it

Select a task and click **Delete**.
